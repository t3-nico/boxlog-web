# SDKs & Libraries

BoxLog provides official SDKs for all major programming languages, making it easy to integrate logging into your applications with just a few lines of code.

## Available SDKs

### Frontend & Backend

#### 🟨 [JavaScript/TypeScript](/docs/sdks/javascript)
*Most popular - Works everywhere*

Full-featured SDK for Node.js backends and modern browsers. TypeScript support included.

```javascript
import { BoxLog } from '@boxlog/sdk'
const logger = new BoxLog({ apiKey: 'your_key' })
logger.info('User logged in', { userId: 'user_123' })
```

#### 🐍 [Python](/docs/sdks/python)
*Perfect for data science and web apps*

Comprehensive Python SDK with Django, Flask, and FastAPI integrations.

```python
from boxlog import BoxLog
logger = BoxLog(api_key='your_key')
logger.info('Processing started', {'job_id': 'job_123'})
```

#### 🐹 [Go](/docs/sdks/go)
*High-performance microservices*

Efficient Go SDK optimized for high-throughput applications and microservices.

```go
import "github.com/boxlog/go-sdk"
logger := boxlog.New("your_key")
logger.Info("Request processed", boxlog.Fields{"duration": 250})
```

#### 💎 [Ruby](/docs/sdks/ruby)
*Rails and Ruby applications*

Ruby SDK with Rails integration and ActiveRecord support.

```ruby
require 'boxlog'
logger = BoxLog.new(api_key: 'your_key')
logger.info('User created', { user_id: 'user_123' })
```

### Additional Languages

## Common SDK Features

### Structured Logging
All SDKs support structured logging with consistent APIs:

```javascript
// JavaScript
logger.info('Event occurred', { userId, action, timestamp })

# Python
logger.info('Event occurred', {'user_id': user_id, 'action': action})

// Go
logger.Info("Event occurred", boxlog.Fields{"userId": userId, "action": action})
```

### Context and Correlation
Track requests across your distributed systems:

```javascript
// Child loggers with context
const requestLogger = logger.child({ requestId, userId })
requestLogger.info('Processing request')
```

### Async and Batching
Optimize performance with batching and async sending:

```javascript
const logger = new BoxLog({
  apiKey: 'your_key',
  batchSize: 100,
  flushInterval: 5000
})
```

### Error Handling
Automatic error serialization and stack trace capture:

```javascript
try {
  riskyOperation()
} catch (error) {
  logger.error('Operation failed', { error, context })
}
```
