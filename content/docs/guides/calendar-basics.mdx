# カレンダーの基本

Dayoptのカレンダーは、あなたの日々の予定とタスクを一目で把握できる中心的な機能です。このガイドでは、カレンダーの基本的な使い方を解説します。

## カレンダービューの種類

Dayoptは3つのカレンダービューを提供しています：

| ビュー | 用途 | ショートカット |
|--------|------|----------------|
| **日表示** | 1日の詳細なスケジュール確認 | `1` |
| **週表示** | 週全体の予定を俯瞰 | `2` |
| **月表示** | 月間の予定概要 | `3` |

ビューの切り替えは、カレンダー上部のボタンまたはキーボードショートカットで行えます。

## 予定を作成する

### クリックで作成

1. カレンダー上の任意の時間帯をクリック
2. 予定作成フォームが表示される
3. タイトル、時間、タグを入力
4. 「保存」をクリック

```
💡 ヒント: 時間帯をドラッグすると、開始〜終了時間を一度に指定できます
```

### キーボードで素早く作成

`N` キーを押すと、現在時刻に新規予定を作成できます。

```typescript
// 予定のデータ構造
interface Plan {
  id: string
  title: string
  startTime: Date
  endTime: Date
  tags: Tag[]
  description?: string
  isAllDay: boolean
}
```

## ドラッグ&ドロップ操作

カレンダー上の予定は、ドラッグ&ドロップで簡単に操作できます：

### 予定の移動

1. 予定をクリックしてホールド
2. 目的の時間帯にドラッグ
3. ドロップして確定

### 時間の調整

予定の上端または下端をドラッグすると、開始時刻または終了時刻を調整できます。

```
⚡ パフォーマンス: ドラッグ中はリアルタイムプレビューが表示されます
```

## タグで予定を整理

予定にタグを付けることで、視覚的に分類できます：

- **仕事**: 青色 🔵
- **プライベート**: 緑色 🟢
- **重要**: 赤色 🔴
- **学習**: 紫色 🟣

```typescript
// タグ付き予定の作成例
const plan = await dayopt.plans.create({
  title: "チームミーティング",
  startTime: new Date("2024-12-23T10:00:00"),
  endTime: new Date("2024-12-23T11:00:00"),
  tags: ["仕事", "ミーティング"]
})
```

## 繰り返し予定

定期的な予定は、繰り返し設定で自動作成できます：

| パターン | 説明 | 例 |
|----------|------|-----|
| 毎日 | 毎日同じ時間 | 朝のルーティン |
| 毎週 | 毎週同じ曜日 | 週次ミーティング |
| 毎月 | 毎月同じ日付 | 月次レポート |

## キーボードショートカット

カレンダーでよく使うショートカット：

| キー | 動作 |
|------|------|
| `N` | 新規予定作成 |
| `T` | 今日に移動 |
| `←` / `→` | 前/次の期間に移動 |
| `1` / `2` / `3` | 日/週/月ビュー切り替え |
| `Delete` | 選択中の予定を削除 |
| `E` | 選択中の予定を編集 |

## モバイルでの操作

スマートフォンでもカレンダーを快適に利用できます：

- **スワイプ**: 左右で日付移動
- **ピンチ**: ズームイン/アウト
- **長押し**: 予定の作成
- **タップ**: 予定の詳細表示
