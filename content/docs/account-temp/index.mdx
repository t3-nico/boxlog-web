## Common Configuration Scenarios

### Small Team Setup
Perfect for startups and small development teams:
- Single workspace with dev/prod environments
- Shared API keys with basic role separation
- Email notifications for critical alerts

### Enterprise Setup
For large organizations with multiple teams:
- Multiple workspaces by team or product
- SSO integration with existing identity providers
- Custom roles and granular permissions
- Audit logging and compliance features

### Multi-Environment Setup
For complex deployment pipelines:
- Separate environments for each stage
- Environment-specific API keys and configs
- Automated deployments with environment promotion
- Environment-specific retention and alerting
