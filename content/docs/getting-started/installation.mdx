# Installation Guide

This guide will walk you through installing and setting up the YourSaaS SDK for your project. We support multiple platforms and frameworks to fit your development needs.

## Prerequisites

Before installing YourSaaS, ensure you have:

- **Node.js 18.0 or later** ([Download Node.js](https://nodejs.org/))
- **npm, yarn, or pnpm** package manager
- **A YourSaaS account** ([Sign up for free](https://yoursaas.com/signup))

## Quick Installation

Choose your preferred package manager:

### npm
```bash
npm install @yoursaas/sdk
```

### yarn
```bash
yarn add @yoursaas/sdk
```

### pnpm
```bash
pnpm add @yoursaas/sdk
```

## Framework-Specific Installation

### Next.js

For Next.js applications, install the optimized package:

```bash
npm install @yoursaas/nextjs
```

Then add the YourSaaS provider to your app:

```tsx
// app/layout.tsx
import { YourSaaSProvider } from '@yoursaas/nextjs'

export default function RootLayout({
  children,
}: {
  children: React.ReactNode
}) {
  return (
    <html>
      <body>
        <YourSaaSProvider
          apiKey={process.env.YOURSAAS_API_KEY}
          projectId={process.env.YOURSAAS_PROJECT_ID}
        >
          {children}
        </YourSaaSProvider>
      </body>
    </html>
  )
}
```

### React

For React applications:

```bash
npm install @yoursaas/react
```

Wrap your app with the provider:

```tsx
// src/App.tsx
import { YourSaaSProvider } from '@yoursaas/react'

function App() {
  return (
    <YourSaaSProvider
      apiKey={process.env.REACT_APP_YOURSAAS_API_KEY}
      projectId={process.env.REACT_APP_YOURSAAS_PROJECT_ID}
    >
      <YourApp />
    </YourSaaSProvider>
  )
}
```

## Environment Configuration

Create a `.env.local` file (or `.env` for Node.js) in your project root:

```bash
# Required
YOURSAAS_API_KEY=your_api_key_here
YOURSAAS_PROJECT_ID=your_project_id_here

# Optional
YOURSAAS_ENVIRONMENT=development
YOURSAAS_BASE_URL=https://api.yoursaas.com
YOURSAAS_TIMEOUT=10000
```

### Getting Your API Keys

1. **Sign in** to your [YourSaaS dashboard](https://dashboard.yoursaas.com)
2. **Create a new project** or select an existing one
3. **Navigate** to Settings â†’ API Keys
4. **Copy** your API key and Project ID

> **Security Warning**: Never expose your API keys in client-side code. Use environment variables and keep them secure.

## TypeScript Support

YourSaaS SDK comes with full TypeScript support out of the box. No additional type packages needed!

```typescript
import { YourSaaSClient, User, CreateUserRequest } from '@yoursaas/sdk'

const client = new YourSaaSClient({
  apiKey: process.env.YOURSAAS_API_KEY!,
  projectId: process.env.YOURSAAS_PROJECT_ID!
})

// Fully typed
const createUser = async (userData: CreateUserRequest): Promise<User> => {
  return await client.users.create(userData)
}
```
