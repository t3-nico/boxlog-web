# 5-Minute Quickstart

Welcome! In just 5 minutes, you'll have a working application that:

- âœ… Creates and authenticates users
- âœ… Makes secure API calls
- âœ… Handles real-time updates
- âœ… Manages user sessions

Let's dive in!

## Step 1: Setup (30 seconds)

First, install YourSaaS and create your project structure:

```bash
npx create-yoursaas-app my-app
cd my-app
npm install
```

Or add to an existing project:

```bash
npm install @yoursaas/sdk
```

## Step 2: Configuration (1 minute)

Create your `.env.local` file with your API credentials:

```bash
# Get these from https://dashboard.yoursaas.com
YOURSAAS_API_KEY=your_api_key_here
YOURSAAS_PROJECT_ID=your_project_id_here
```

> **Need API keys?** [Sign up for free](https://dashboard.yoursaas.com) and get your keys instantly.

## Step 3: Initialize the Client (30 seconds)

Create a client instance:

```typescript
// lib/yoursaas.ts
import { YourSaaSClient } from '@yoursaas/sdk'

export const client = new YourSaaSClient({
  apiKey: process.env.YOURSAAS_API_KEY!,
  projectId: process.env.YOURSAAS_PROJECT_ID!
})
```

## Step 4: Create Your First User (1 minute)

Let's create a simple user registration:

```typescript
// pages/api/register.ts (or app/api/register/route.ts for App Router)
import { client } from '@/lib/yoursaas'

export async function POST(request: Request) {
  try {
    const { email, password, name } = await request.json()
    
    // Create user with YourSaaS
    const result = await client.auth.register({
      email,
      password,
      name
    })
    
    return Response.json({
      success: true,
      user: result.user,
      session: result.session
    })
  } catch (error) {
    return Response.json({
      success: false,
      error: error.message
    }, { status: 400 })
  }
}
```

## Step 5: Add Authentication (1 minute)

Create a login form component:

```tsx
// components/LoginForm.tsx
'use client'
import { useState } from 'react'

export function LoginForm() {
  const [email, setEmail] = useState('')
  const [password, setPassword] = useState('')
  const [user, setUser] = useState(null)

  const handleLogin = async (e: React.FormEvent) => {
    e.preventDefault()
    
    const response = await fetch('/api/auth/login', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ email, password })
    })
    
    const data = await response.json()
    
    if (data.success) {
      setUser(data.user)
      // Store session token
      localStorage.setItem('token', data.session.token)
    }
  }

  if (user) {
    return (
      <div className="p-6 bg-green-50 rounded-lg">
        <h2 className="text-xl font-bold text-green-800">
          Welcome back, {user.name}! ðŸŽ‰
        </h2>
        <p className="text-green-600">You're successfully logged in.</p>
      </div>
    )
  }

  return (
    <form onSubmit={handleLogin} className="space-y-4 max-w-md mx-auto">
      <div>
        <label className="block text-sm font-medium mb-1">Email</label>
        <input
          type="email"
          value={email}
          onChange={(e) => setEmail(e.target.value)}
          className="w-full p-2 border rounded focus:ring-2 focus:ring-blue-500"
          required
        />
      </div>
      
      <div>
        <label className="block text-sm font-medium mb-1">Password</label>
        <input
          type="password"
          value={password}
          onChange={(e) => setPassword(e.target.value)}
          className="w-full p-2 border rounded focus:ring-2 focus:ring-blue-500"
          required
        />
      </div>
      
      <button
        type="submit"
        className="w-full bg-blue-600 text-white p-2 rounded hover:bg-blue-700"
      >
        Login
      </button>
    </form>
  )
}
```

## Step 6: Test Everything (30 seconds)

Start your development server and test:

```bash
npm run dev
```

Visit `http://localhost:3000` and try:

1. **Register a new user** with email/password
2. **Login** with those credentials  
3. **View the welcome message**
