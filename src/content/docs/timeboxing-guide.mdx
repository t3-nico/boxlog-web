# タイムボクシングガイド

このページは `timeboxing-guide.mdx` です。

## タイムボクシングとは

タイムボクシングは、カレンダー上の特定の時間枠（タイムボックス）を特定のタスク専用に確保する時間管理戦略です。

## BoxLogでの実践方法

1.  カレンダービューを開きます。
2.  タスクを割り当てたい時間帯をドラッグします。
3.  タスクの詳細を入力し、保存します。
