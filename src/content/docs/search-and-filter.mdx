# 検索とフィルタ機能

このページは `search-and-filter.mdx` です。

## 強力な検索機能

画面上部の検索バーから、すべてのタスクを対象にキーワード検索を実行できます。
タスク名だけでなく、詳細内容も検索対象となります。

## フィルタの活用

- **タグでフィルタ**: 特定のタグが付いたタスクのみを表示します。
- **日付でフィルタ**: 期限が今日、今週、または特定の日付範囲内のタスクを絞り込みます。
- **担当者でフィルタ**: （チーム機能）特定のメンバーに割り当てられたタスクを表示します。
