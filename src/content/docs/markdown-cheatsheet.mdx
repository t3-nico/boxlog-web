## テキストの装飾

**太字 (Bold)**
*イタリック (Italic)*
~~打ち消し線 (Strikethrough)~~
`インラインコード (Inline Code)`

## 箇条書きリスト（順序なし）

-   項目1
-   項目2
    -   ネストされた項目2.1
    -   ネストされた項目2.2
-   項目3

## 番号付きリスト（順序あり）

1.  最初の項目
2.  二番目の項目
3.  三番目の項目
    1.  ネストされた項目3.1
    2.  ネストされた項目3.2

## 引用

> これは引用ブロックです。
> 他の人の言葉や、ドキュメントからの抜粋などを記述するのに使います。
>
> > ネストされた引用も可能です。

## コードブロック

```javascript
// Javascriptのコードブロックサンプル
function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet('World');
```

```python
# Pythonのコードブロックサンプル
def add(a, b):
    return a + b

print(add(5, 3))
```
